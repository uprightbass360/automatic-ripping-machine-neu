ARM Setup & Health Check - Brainstorming

Which improvement matters most for first-time users?

A

Improved Setup Wizard (API Keys)

Add an explicit step for OMDB/TMDB/TVDB key entry with live validation during first-run setup. Users currently discover broken metadata after their first rip fails to identify.

B

Permissions Health Check

A diagnostic panel showing volume ownership, path writability, and UID alignment across ARM/transcoder/UI. Users currently have to read container logs to diagnose permission errors.

C

Both - Scoped Together

Add API key step to the wizard AND a permissions health panel. Design them as one feature since both serve the "first-run success" goal.

1

Setup Wizard Only

First-run experience only. After setup is complete, users go to Settings for changes.

2

Always Accessible

Available in the setup wizard AND as a dashboard/settings panel for ongoing diagnostics.
